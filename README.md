# javajungle
This repository will include all the Java projects I have worked on and completed
The headings below indicate the year projects were made (or came into GitHub)
## 2026
### Next Date Occurence
I've had this weird fascination for calendar dates. One of these is randomly figuring out when is the next occurence of X calendar date. For example, when is my birthday next on a Monday? That is what has led to this idea coming to fruition.

## 2025
### DLS
This is a simple program, where I attended to calculate DLS percentages for rain-affected games in limited-overs cricket (a lot of my initial programs are on the sport).

### ICC
This was a simple program for key details of ICC events: Host Nations, Player of the Tournament, of the Final, Winning Team, Runners-Up. This covers all men's and women's senior events, so the World Cups (ODI/T20), Champions Trophy (women's first edition from 2027), and Test Championship (men only).

### NRR
This was I first calculated Net Run Rate, which is how team progress is measured, and qualfication for further stages are decided (particularly for 2 teams tied on points in a group)

### Snakes & Ladders
I made a Snakes and Ladders game, based on a personal board I have. It is coded to how I play. If you get 3 rolls of six, it's nullified!

### T20 World Cup Quiz
This is a quiz game all about the Men's and Women's T20 World Cups!
